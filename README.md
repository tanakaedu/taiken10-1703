# サンプルゲーム
キャラクターやオブジェクトを作ってもらって、今後の作品に組み込む。

## キャラクター
- モデル
- アイテムを持たせる
- 名前とメッセージを仕込む


## 目的
- 作成したキャラクターを動かして楽しめるもの
- パーティクル、ポストエフェクトで調整できるもの

## アニメーション
- 立ち、歩き、走り、ジャンプ、ダメージ、倒れ、ポーズ

## 要素
- 手前を向く動作がある
- それなりにアップ
- ブロックでステージ作り

- 敵
- 爆弾
- 集める
